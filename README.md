## 📚 About

`while(!success) { study(); }`는 CS(Computer Science) 핵심 개념들을 함께 공부하고,  
**웃고, 열정 불태우며**, 결국 **성공에 도달**하기 위한 스터디입니다.  

혼자 공부할 땐 `segmentation fault` 나던 사람도,  
같이 하면 어느새 `return 0;`을 찍게 됩니다.
